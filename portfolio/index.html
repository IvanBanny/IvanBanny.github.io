My "Assignments app":

Online demo Github repo

Simple Electron app that I made for managing assignments: you can add new ones and erase ones that are completed by dragging the slider. Assignments are sorted by date and, if date is similar, by level. Level also affects the color of assignment description background. Level 5 (the hardest / most important) has red background and level 1 (the easiest / least important) has green background. (2022)

My "Visual Sort":

Video (3.5 min)

I made an app for visualizing sorting algorithms using Visual Studio .NET. It also shows the bars that are swapped. You can control the speed and the number of bars. There are 6 different sorting algorithms. As a bonus, there is a hidden portrait of a famous Ukrainian Alexander Avramenko in the corner. When clicked upon 5 times – 5 candles are placed where clicks happened and the portrait appears. When mouse hovers above it – he smiles. (2021)

My "Testing System":

Video (2 min)

Testing system “Avramenko” that I made using Visual Studio .NET: Simple testing system that scans a folder “Tests” and transforms .txt files into tests that you can take. You can pick the test you want to take. In the test.txt there is a number of seconds specified – time for test. After it expires – test stops and the result is displayed. The possible answers are displayed in a random order. There are 2 types of questions: boxes and radio buttons. The type is specified for each question in .txt file. As a bonus, there is a portrait of a famous Ukrainian Alexander Avramenko in the corner. Each time an incorrect answer is given – he becomes more angry. (2021)

My "Calculator":

Video (2 min)

I also made a simple calculator in Visual Studio .NET that has some basic and somewhat more advanced operators. It does not break when 2 operators are given in row, when "=" is not placed, if there was no operator before "=" or if multiple "=" go in row. It's very hard to brake it because I fixed most of the bugs. When division by 0 happens – it displays “No, thanks”. (2021)
